# Finanzas y administracion Febecos

Carpeta de trabajo para reunir documentacion contable, administrativa y financiera.

## Como cargar archivos

Sugerencia de orden inicial:

- `01-bancos/`: extractos, movimientos, saldos, comprobantes bancarios.
- `02-ventas-cobros/`: facturas emitidas, recibos, pagos de clientes, cuentas por cobrar.
- `03-compras-pagos/`: facturas de proveedores, ordenes de pago, cuentas por pagar.
- `04-impuestos-contable/`: IVA, IIBB, ganancias, balances, reportes del contador.
- `05-sueldos-rrhh/`: recibos, cargas sociales, pagos de personal.
- `06-stock-operaciones/`: inventario, compras de materiales, movimientos operativos.
- `99-referencias/`: criterios, notas, capturas, documentos explicativos.

Tambien se pueden copiar archivos sueltos aca al principio. Despues los ordenamos.

## Objetivo

Construir una base administrativa para:

- identificar ingresos, egresos, deudas y saldos;
- detectar faltantes o documentos pendientes;
- armar un flujo de trabajo administrativo;
- alimentar un dashboard financiero de Febecos;
- preparar datos para analisis con IA y reportes periodicos.
